# WPS-SDK-SPRING-BOOT-STARTER-COLLECT

This is a Spring Boot Starter project collection library that focuses on collecting, organizing,
and providing resources related to WPS Office integration development.

## wps-sdk-spring-boot-starter

WPS SDK Spring Boot Starter is an open-source core component project focused on simplifying the integration of WPS
Office and
Spring Boot applications.

It aims to become the preferred toolkit for Spring Boot developers to quickly access WPS document services (preview,
edit, convert, etc.).

## wps-sdk-spring-boot-starter-test

WPS SDK Spring Boot Starter Test is a supporting testing and development assistance module for the WPS SDK Spring Boot
Starter project.

It aims to provide a convenient and efficient environment for developers using WPS SDK Spring Boot Starter to quickly
verify configurations, test integration features, understand API call processes, and simulate WPS server behavior.
